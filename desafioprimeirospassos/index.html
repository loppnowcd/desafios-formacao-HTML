<h1 até h6>

CABEÇALHOS, OU HEADERS

Os cabeçalhos são títulos de novas seções. Começam do h1 e vão até o h6. O h1 é o maior, e o h6 é o menor. Vai diminuindo do 1 para o 6. voltar

<p>

PARÁGRAFO

Os parágrafos funcionanam igualzinho em um livro, quando queremos separar uma nova linha de pensamentos. Às vezes também podemos usar parágrafos para ir separando textos e ficar mais bonitinho e não ficar tudo em um único bloco. voltar

<mark>

MARCAÇÃO

A tag mark serve para destacar um texto como se houvesse sido usada uma caneta marca texto. As propriedades da marcação podem ser alteradas em arquivo CSS. voltar

<small>

TEXTO PEQUENO

A tag <small> serve para diminuir um texto como se ficasse reduzido, menor. Está sendo utilizado, por exemplo nos botões 'voltar'. voltar

<strong>

TEXTO FORTE

A tag <strong> serve para deixar o texto mais forte, igual negrito. Está sendo utilizado nesta página em vários títulos e cabeçalhos pra dar mais importância. voltar

<ol e ul>

LISTAS

As tags <ol e ul> servem para criar uma lista. OL é de lista ordenada (números, sequência), enquanto UL é de lista não ordenada. O cabeçalho desta página foi construída com um UL. voltar

<li>

ITENS DE LISTAS

A tag <li> serve para criar os itens dentro das listas OL ou UL. Cada item é um <li>. voltar

<a>

ÂNCORAS OU LINKS

A tag <a> é para criar âncoras ou links. Vamos lembrar que o link pode ser dentro da própria página (move você para o texto que você quer encontrar), ou pode ser link externo. O link externo pode abrir nesta aba do navegador (como se recarregasse a página), ou pode abrir em outra aba (a página atual contiunua aberta). Seguem exemplos:

<a> href="https://www.example.com"</a> Acessar na mesma aba

<a> href="https://www.example.com" target="_blank"</a> Acessar em uma nova aba voltar

<hr>

SEPARADORES

A tag <hr> serve para criar um traço horizontal na tela. Serve como uma espécie de separação. Essa é uma tag auto fechada, não tem necessidade de fechá-la. Estamos usando um <hr> entre todas as seções desta página. voltar

<sub> e <sup>

SUBESCRITO E SOBESCRITO

As tags &lsub> e &lsup> servem para colocar o texto logo abaixo ou logo acima. Algumas vezes são utilizados para fórmulas matemáticas, textos de química, etc.

<blockquote>

REFERÊNCIAS

A tag <blockquote> é usada para destacar uma citação de outra fonte, geralmente um texto mais longo. Essa tag indica que o texto dentro dela é uma citação e geralmente é exibido com um recuo à esquerda, diferenciando-o do restante do conteúdo na página. O navegador geralmente estiliza o texto dentro da <blockquote> com um recuo, mas posso personalizar a aparência com CSS se desejar.

<del>

TEXTO EXCLUÍDO

A tag <del> serve para indicar que um texto foi excluído, e aparece riscado. voltar

<font>

FONTE

A tag <font> era usada em HTML para definir a aparência do texto, como a fonte, o tamanho e a cor. No entanto, é importante notar que a tag <font> é obsoleta nas versões mais recentes do HTML (HTML5) e não é recomendada para uso. Em vez disso, recomenda-se que o estilo do texto deve ser controlado através de CSS. voltar

<abbr>

ABREVIAÇÃO

A tag <abbr> é usada para representar uma abreviação ou um acrônimo. Ela fornece uma maneira semântica de marcar texto que é uma forma reduzida de uma palavra ou frase, permitindo que os navegadores ou leitores de tela reconheçam a natureza do texto e, potencialmente, exibam a forma completa. Quando o usuário passa o mouse sobre a abreviação, geralmente uma dica de ferramenta (tooltip) aparece com a explicação.

A Organização das Nações Unidas é frequentemente abreviada como ONU.
